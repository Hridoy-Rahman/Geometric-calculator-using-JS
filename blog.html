What is DOM and What is the purpose of using DOM ?

Answer : The Full meaning of DOM is Document Object Model. It represents the document as object. The purpose of using DOM:

  • To access the element of html
  • To get,Set and remove classes
  • To manipulate remove and update styles

How will you select HTML elements using DOM? Name the DOM methods.

We can select HTML element using DOM methods. The way of selecting HTML element is document.DOM-selector("id")


Name of the DOM Methods:

  1. getElementById()
  2. getElementsByClassName()
  3. getElementsByTagName()
  4. querySelector()
  5. querySelectorAll()

What is event bubble?

Event bubble is a term of DOM.When an element receives an event like click,to blur,to scroll etc then it happens.It bubbles up to its parent and grandparent untill it gets its root element,when it happens.The representation on an event bubble looks like a tree.

What is a callback function and why will you use it?

A function that passed as an arguement of another function is called callback function.This function executes after the main function's execution.
